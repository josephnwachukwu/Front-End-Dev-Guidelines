What exactly is a directive

Why Use a directive?

directives are what angular is built on. mostly all angular functionality is attached via the directive. Directives allow you to extend the functionality of an element using javascript. Directives also allow you to create your own elements and transform them on the page.

Setting up a directive

Markup
	app.directive('formatNumber' function(){
		return {
			link: function(scope,elem,attrs) {
				$(elem).plugin();
			}
		};
	});
restrict

scope

The scope property allows you to define whether or not you want to have a new scope screated for the directive or if you want to use the parent scope. scope is returned as a boolean

template

transclude

the transclude property allows you to replace the content of an element where the directive is placed. usually when you transclude an element you provide a template that you will use to replace the current markup. transclude is returned as a boolean

	app.directive('formatNumber' function(){
		return {
			transclude:false,
			link: function(scope,elem,attrs) {
				$(elem).plugin();
			}
		};
	});
